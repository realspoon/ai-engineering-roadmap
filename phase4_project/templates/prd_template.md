## 4. 비기능 요구사항

### 4.1 성능
- 응답 시간: 평균 5초 이내
- 동시 사용자: 최소 10명

### 4.2 가용성
- 가동률: 99% 이상
- 장애 복구 시간: 1시간 이내

### 4.3 보안
- 인증: API 키 기반
- 데이터 암호화: 전송 중 TLS
