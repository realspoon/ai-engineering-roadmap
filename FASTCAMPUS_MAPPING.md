## 🔥 추천 학습 순서 (FastCampus 기준)

### 필수 선행 (Phase 1 시작 전)
1. Python 기초 (복습 필요시)
2. ChatGPT 완벽 가이드
3. 프롬프트 엔지니어링 완벽 가이드

### Phase 1-2 병행
4. OpenAI API 완벽 가이드
5. LangChain으로 AI 에이전트 개발
6. RAG 시스템 구축
7. AI 에이전트 완벽 가이드

### Track 선택 후
- **Track A**: Python 데이터 분석 → 데이터 시각화 → Streamlit
- **Track B**: SQL 기초 → SQL 데이터 분석 → SQL 완벽 가이드
- **Track C**: 데이터 엔지니어링 → Python 데이터 분석

### 프로젝트 단계
8. FastAPI
9. Streamlit
10. AI 서비스 배포
