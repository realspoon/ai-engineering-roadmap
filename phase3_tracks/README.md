## 📂 트랙별 상세

### [Track A: Data Analyst Agent](./track_a_data_analyst/)

**목표**: 자연어로 데이터 분석을 요청하면 자동으로 분석하고 시각화 및 인사이트를 생성하는 Agent

| 주차 | 주제 | 핵심 기술 |
|------|------|----------|
| Week 17 | EDA 자동화 | pandas profiling, 통계 분석 |
| Week 18 | 시각화 자동화 | matplotlib, plotly 코드 생성 |
| Week 19 | 인사이트 생성 | Code Interpreter, 자동 리포트 |
| Week 20 | 분석 파이프라인 | 질의 의도 파악, 분석 계획 |
| Week 21 | 고급 분석 | 시계열, 상관관계, 예측 |
| Week 22 | 멀티모달 | CSV, Excel, PDF, 이미지 |
| Week 23 | UI 구축 | Streamlit 대화형 인터페이스 |
| Week 24 | 통합 테스트 | Agent v1.0 완성 |

---

### [Track B: Text-to-SQL Agent](./track_b_text_to_sql/)

**목표**: 자연어 질문을 SQL로 변환하여 데이터베이스를 쿼리하고 결과를 해석하는 Agent

| 주차 | 주제 | 핵심 기술 |
|------|------|----------|
| Week 17 | SQL 고급 | Window Functions, 최적화 |
| Week 18 | 스키마 분석 | 메타데이터 추출, ERD |
| Week 19 | Text-to-SQL 기초 | 프롬프트 엔지니어링 |
| Week 20 | Agent 구현 | 스키마 인식, 쿼리 검증 |
| Week 21 | 고급 쿼리 | Multi-step, 분해 |
| Week 22 | RAG 스키마 검색 | 임베딩, Few-shot |
| Week 23 | 결과 해석 | 자연어 설명, 시각화 |
| Week 24 | 통합 테스트 | Agent v1.0 완성 |

---

### [Track C: Data Quality Agent](./track_c_data_quality/)

**목표**: 데이터 품질을 자동으로 검사하고 이상을 탐지하며 알림을 발송하는 Agent

| 주차 | 주제 | 핵심 기술 |
|------|------|----------|
| Week 17 | 품질 개념 | 6가지 차원, 프레임워크 |
| Week 18 | 프로파일링 | Great Expectations, 검증 |
| Week 19 | 이상 탐지 | 통계, ML, 시계열 |
| Week 20 | LLM 품질 검사 | 의미적 검증, 자연어 규칙 |
| Week 21 | 실시간 모니터링 | 스트리밍, 대시보드 |
| Week 22 | 자동 해결 | RCA, 수정 제안 |
| Week 23 | 리니지 분석 | 영향 분석, 전파 추적 |
| Week 24 | 통합 테스트 | Agent v1.0 완성 |
